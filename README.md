# Fun-Game — x86 組合語言小遊戲

使用 x86 Assembly 組合語言撰寫終端機遊戲，
從而設計底層程式、暫存器操作、流程控制與記憶體存取。

 1. **遊戲一 - 猜數字**：1-100 範圍的猜數字小遊戲， 似終極密碼，只是不會告訴使 用者下一 次範圍，僅提示數字 是過大還是過小。
 2. **遊戲二 - 貪食蛇**：一條細長的蛇畫面，玩家使用上下左 右鍵盤操作，它會一直前進，並要避 免觸碰到牆。貪食蛇每吃掉一個食物 ，它的身體便增長一些。當吃過超過 十次 (累計10分以上)，蛇的移動速度 逐漸加快，讓遊戲的難度漸漸變大。

---

## 專案簡介

Fun-Game 是以組合語言實作的文字型遊戲，
透過鍵盤輸入與畫面輸出互動呈現：

* CPU 暫存器與堆疊操作
* 條件跳轉與流程控制
* 資料段 / 程式段 / 堆疊段應用
* 底層 I/O 與字元處理

---

## 使用技術

| 類型        | 技術                        |
| --------- | ------------------------- |
| Language  | x86 Assembly              |
| Assembler | MASM / TASM / NASM |
| Platform  | DOS / Windows Console     |
| I/O       | BIOS / DOS Interrupt      |

---

## 功能說明

* 使用鍵盤輸入控制遊戲流程
* 終端機畫面即時更新
* 透過條件判斷控制遊戲邏輯
* 可重複遊玩與重新開始
